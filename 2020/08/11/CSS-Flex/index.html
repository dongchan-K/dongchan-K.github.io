CSS Flex

Flex는 레이아웃에 적합한 1차원 정렬 방식이다

Flex는 Container와 Items의 2개의 개념으로 나뉜다

  • 정렬할 요소들의 부모 요소에 display: flex; 적용시 부모 요소는 Flex Container, 정렬할 요소들은 Flex items가 된다

Container에서 사용 가능한 속성

  • display:flex : Flex Container를 정의

  • flex-flow(단축 속성) : Flex Items의 주 축(direction), 줄 바꿈(wrap)을 설정

  • justify-content : 주 축의 정렬 방식을 선택

  • align-content : 교차 축의 정렬 방식을 선택 (items가 1줄일 경우)

Items에서 사용 가능한 속성

  • order : Flex Items의 순서를 정렬

  • Flex(단축 속성) : Flex Item의 증가 너비 비율(grow), 감소 너비 비율(shrink), 기본 너비 설정(basis)

  • align-self : 교차 축에서 Items가 아닌 Item의 개별 정렬 방법을 설정 (align-items보다 우선한다)
